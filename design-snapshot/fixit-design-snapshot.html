FixIt Campus — Current UI Snapshot

Generated 2026-07-18 from the live codebase (React + Vite + Tailwind, Supabase backend). ~50 screens across Student / Staff / Admin roles. Purpose: source material for a full UI/UX redesign in Google Stitch.

How to use this with Stitch: Stitch reads text prompts and reference images — it does not import HTML or code. Workflow:
  1. Start a Stitch project → paste the Master Style Prompt (Section 1) as your first message so every screen shares the design language.
  2. For each screen, paste that page's prompt block (copy button on each).
  3. Optional but recommended: run npm run dev, screenshot each real page, and upload the screenshot alongside the prompt — Stitch uses images as layout reference.

1. Master Style Prompt (paste into Stitch first)

Design a web app called "FixIt Campus" — a university super-app for BUBT (Bangladesh University of Business & Technology). Modern, clean, professional SaaS dashboard style. DESIGN SYSTEM: - Font: Plus Jakarta Sans (weights 400–800). Compact type scale: body 13px, small labels 11–12px, page titles 26–32px extrabold with tight letter-spacing. - Primary brand color: #2b5be3 (blue). Hover/darker: #1f47c4. Light tints: #eef3ff and #dde7ff. - Light theme: page background #f5f7fb, cards white #ffffff, secondary surface #eef2f8, borders #e4e9f1, text #0f1a2e, secondary text #46536e, muted text #8693aa. - Dark theme exists too: background #0a0f1c, cards #111829, borders #232f48, text #e9eefb, brand becomes #6a8cf2. - Status colors: success #12915e (bg #e3f5ec), warning #b9760a (bg #fbefdb), danger #d63d35 (bg #fbe7e5), info = brand blue. - Each feature has its own accent used ONLY for icon tiles and category badges (chrome stays brand blue everywhere): Reports #4f6bed, Lost&Found #c77d1a, Clubs #8b5cf0, Events #e0568a, Jobs #0e9c8a, Announcements #3e7de0, Study #2ba0c9, Bus #e08a2b, Medical #e2483d, Marketplace #2e9e63, RideShare #6e8b1f, Blood #c7344a, Prayer #1f8a5b, Calendar #d4553a, Routines #5c6bc0, CoverPage #00838f. - Corner radii generous: buttons/inputs 14px, cards 18px, pills fully rounded. Soft subtle shadows. - Buttons: primary = solid brand blue, white bold text, 44px tall; secondary = white with border; destructive = solid red; ghost = text only. Icons (Lucide style) on most buttons. - Cards: white, 1px light border, 18px radius, small shadow, 20px padding. - Badges: small rounded-full pills with tinted backgrounds. - Stat cards: small colored icon tile (rounded square) + big bold number + muted label. - Empty states: centered icon, bold title, muted one-line message, action button. - Forms: label above input, 48px tall inputs, light border, brand focus ring, red inline errors. APP LAYOUT (all signed-in screens): - Fixed left sidebar 240px, white, logo at top; grouped nav ("Campus Life", "Services", "Community" section labels in 11px uppercase); nav items = icon + label, rounded, active item has light-blue background + brand text; "Log out" pinned at bottom. On mobile it becomes a slide-in drawer. - Sticky translucent top bar (blurred white, 64px): page title left; right side = theme toggle, notification bell with red unread badge, user name + email + avatar, role badge (Student blue / Staff amber / Admin green), logout icon. - Content column centered, max-width 1152px, generous padding. - Every page starts with a big bold page title (26–32px) + one-line muted subtitle, with a primary action button on the right.

Typography

  • Font: Plus Jakarta Sans (400 / 500 / 600 / 700 / 800) + Hind Siliguri for Bengali glyphs (line-height 1.6).
  • Compact scale: xs 11px · sm 12px · base 13px · md 14px · lg 15px · xl 16px · 2xl 18px · 3xl 21px · 4xl 26px · 5xl 32px.
  • h1 letter-spacing −0.02em, h2 −0.01em. Page titles bold/extrabold; section labels 11px bold UPPERCASE tracking +0.06em muted.

Radii & shadows

  • Radii: xs 8 · sm 10 · md 14 (buttons/inputs) · lg 18 (cards) · xl 22 · 2xl 26 · 3xl 32 · full 999 (pills/avatars).
  • Shadows (very soft, navy-tinted): sm 0 1px 2px rgba(15,26,46,.05) · md 0 2px 6px .06 · lg 0 6px 16px .10 · xl 0 12px 32px .18.

3. Component library (shared across every page)

  • Button — primary (solid brand, white bold text), secondary (white + border), destructive (solid red), ghost (text). md 44px / sm 36px tall, radius 14px, Lucide icon left or right, spinner replaces icon when loading.
  • Input / Select / Textarea — 48px tall, white, 1px border #e4e9f1, radius 14px, brand focus ring; password gets eye toggle; error = red border + red 11px message with alert icon. Label above (14px semibold, red * when required), optional muted hint below.
  • Card — white, 1px border, radius 18px, shadow-sm.
  • Badge — rounded-full pill, tinted bg + strong text (blue/amber/emerald/red…), optional icon. Used for statuses (Open amber, In Progress blue, Resolved green), roles, categories.
  • StatCard — card with colored rounded icon tile (40–44px), big bold number (26px), muted label.
  • AccentTile — rounded-square icon chip: feature accent at 15% opacity bg + solid accent icon.
  • FilterTabs — pill tab row with per-tab count bubbles (e.g. All · Open · In Progress · Resolved).
  • SegmentToggle — 2-option segmented control (e.g. "To Campus / From Campus", "Today / This month").
  • SearchField — input with left search icon, placeholder "Search …".
  • EmptyState — centered big muted icon, bold title, one-line message, optional action button.
  • Avatar — circular photo, falls back to initials on colored bg.
  • Modal — centered card, icon + title + description, footer buttons; dimmed backdrop.
  • Toast — corner notification: success / error with icon.
  • CountdownBanner — hero strip (Bus/Prayer): soft tinted rounded-xl panel, 48px accent tile, uppercase eyebrow, 21–24px bold title, live countdown ("in 1h 05m") + meta line.
  • DataTable — bordered rounded table, uppercase 11px muted header row on surface-2, row hover.
  • PageHeader — 26–32px bold title + muted subtitle, action button(s) right-aligned.
